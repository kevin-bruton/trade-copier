I would like to build a trade copier in python with a UI in dark mode. The trade copier should be able to copy trades between instances of MetaTrader 4 and MetaTrader 5. It should used a socket interfact to communicate between instances and an EA that is installed on each MetaTrader instance that it is to interact with. The Python App will be the socket server and  the MT instances (whether MT4 or MT5) will be the socket clients. Each EA will report its broker, account number, balance, equity, and whether it is a demo or a real account, and any  other relevant information. The Python App will allow the user to configure the which instance is to copy to which other instance, which magic numbers are to be copied (or all) amoung the available magic numbers in the source instance, and also provide an instrument mapping in case the symbols are different between brokers. This configuration should be saved to a   yaml file in the project for easy referencing and so it can be reloaded on startup. It should also allow the user to configure position size in the destination MT instance, by fized   size by magic number or by a percentage of the account or by a fixed dollar amount. Trades should be copied, only when a position is opened, with its set stop loss and take profit, not pending orders. If the trade is closed at the source MT before a stop loss or take profit is hit, it should also be closed on the destination MT. You can use, as a reference, the     files in the mql_reference folder, and use the header files in the mql folder for the implementation. Do not use the files in mql_reference as a part of the implementation - only for  reference of the approach to be used. Consider this carefully, and explore possible edge cases and risks while live trading. Based on this, develop an implementation plan, for the EAs that will have to be installed in the MetaTrader instances and for the Python App. It should be visible in the MT instance whether it is connected or not. It should also be clear in   the Python App UI what MT instances are connected and what MT instances are being copied from and what MT instances are being copied to.
